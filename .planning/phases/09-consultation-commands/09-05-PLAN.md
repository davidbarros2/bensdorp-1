| audit_log.payload (JSON) → terminal | parsed with json.loads (never eval); only known keys extracted; fallback truncates to 60 chars |
| audit_log.symbol → terminal | uppercased S&P symbol or NULL — rendered via render_table |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-05-T1 | Tampering | --since / --until SQL injection | mitigate | Strict `date.fromisoformat` parse → datetime bindparam; ValueError → print_error + exit code 1 |
| T-09-05-T2 | Tampering | --symbol SQL injection | mitigate | Parameterized `audit_log.c.symbol == symbol.upper()` — no string interpolation |
| T-09-05-T3 | Tampering | --type validation | mitigate | `Optional[AuditEventType]` Typer annotation auto-rejects unknown values (RESEARCH.md security note); StrEnum value space is fixed |
| T-09-05-T4 | Tampering | payload JSON injection | mitigate | `json.loads` (never eval); parse failure → safe truncation `[:60]`; only specific known keys extracted, not whole-dict rendered |
| T-09-05-T5 | Tampering | render_table cell markup | mitigate | render_table applies markup-safe output (ui/tables.py — verified Phase 5) |
| T-09-05-D1 | Denial of Service | --limit large value | accept | Single-user CLI; user controls own limit |
| T-09-05-V1 | Tampering | invalid date input | mitigate | try/except ValueError + typer.Exit(code=1) |
| T-09-05-SC | Tampering | npm/pip/cargo installs | accept | No new packages |
