| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-02 | Tampering | `DATA_DIR = Path(env)` | accept | Path strings are validated by downstream consumers (Phase 6 `init`) when directory creation is attempted; v1 is single-user, single-machine, low value. Phase 5 only stores the Path. |
| T-05-03 | Information Disclosure | `format_*` formatters | accept | All formatters operate on numeric/datetime inputs; no PII, no secrets pass through them. Outputs are display strings only. |
| T-05-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages installed in this plan or phase. RESEARCH.md §Package Legitimacy Audit confirms zero new dependencies. |
</threat_model>
