    - In inner finally: backup_sqlite_conn.close()
    - In outer finally: raw_conn.close()
    - After try/finally: shutil.copy2(backup_path, latest_path) — updates latest.db unconditionally using file copy (NOT symlink; Windows [WinError 1314] confirmed)
